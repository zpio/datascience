Histogramas con GGPLOT2¶

Es relativamente sencillo construir un histograma con ggplot2 gracias a la función geom_histogram().

Solo se necesita una variable numérica en la entrada.

Tenga en cuenta que se activa un mensaje de advertencia: debemos ocuparnos del ancho del intervalo.

Histograma básico con geom_histogram¶

In [1]:
# library
library(tidyverse)
library(ggplot2)
 
# dataset:
data=data.frame(value=rnorm(100))

# basic histogram
p <- ggplot(data, aes(x=value)) + 
  geom_histogram()

Histograma de varios grupos con facet_wrap()¶

Si el número de grupo que necesita representar es alto, dibujarlos en el mismo eje a menudo resulta en una figura desordenada e ilegible.

Una buena solución es usar un múltiplo pequeño donde cada grupo se represente en una fracción de la ventana, lo que hace que la figura sea fácil de leer. Esto es bastante fácil de construir gracias a la función facet_wrap() de ggplot2.

In [13]:
options(warn=-1)

#install.packages("viridis")

library(hrbrthemes)

library(forcats)

# creamos y limpiamos un data set
data4 <- read.table("https://raw.githubusercontent.com/zonination/perceptions/master/probly.csv", header=TRUE, sep=",")
data4 <- data4 %>%
  gather(key="text", value="value") %>%
  mutate(text = gsub("\\.", " ",text)) %>%
  mutate(value = round(as.numeric(value),0))

p6 <- data4 %>%
  mutate(text = fct_reorder(text, value)) %>%
  ggplot( aes(x=value, color=text, fill=text)) +
    geom_histogram(alpha=0.6, binwidth = 5) +
    scale_fill_viridis(discrete=TRUE) +
    scale_color_viridis(discrete=TRUE) +
    theme_ipsum() +
    theme(
      legend.position="none",
      panel.spacing = unit(0.1, "lines"),
      strip.text.x = element_text(size = 8)
    ) +
    xlab("") +
    ylab("Assigned Probability (%)") +
    facet_wrap(~text)
